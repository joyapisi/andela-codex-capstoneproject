# Refugee Skills Wallet

A hackathon MVP for a low-bandwidth, multilingual digital profile platform for displaced people and NGOs/community organizations.

The app lets someone create a lightweight skills profile without formal documents, preview it, and view NGO-facing summaries generated by a mocked AI layer. It is intentionally simple and ready for future Supabase, OpenAI, Whisper, offline sync, and secure sharing integrations.

## Project Structure

```text
apps/
  backend/      Node.js + Express API
  frontend/     React + Vite app
packages/
  shared/       Shared TypeScript profile types
```

## Requirements

- Node.js LTS
- npm

## Setup

```bash
npm install
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:4000

You can also run each app separately:

```bash
npm run dev:frontend
npm run dev:backend
```

## MVP Features

- Landing page for the project idea
- Create/edit profile page backed by local browser storage
- Profile preview page with mock translation
- NGO/community dashboard with mock AI summaries
- Express API with in-memory profile data
- Placeholder service files for future OpenAI, Whisper, and Supabase work

## Next Steps

- **OpenAI:** Replace `mockAiService` with real profile summarization and translation calls.
- **Whisper:** Add voice-to-text capture to help users describe experience in their preferred language.
- **Supabase:** Replace in-memory backend storage with authenticated row-level profile storage and sync metadata.
- **Offline sync:** Add a service worker, background sync queue, and conflict-safe profile merge behavior.
- **Secure sharing:** Add QR codes and expiring share links that expose only user-selected profile fields.
